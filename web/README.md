# ReportGen Web

Next.js app for tax return OCR → integrator Excel paste.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy (OVH VPS)

Production hosting is Docker on OVH. See **[deploy/vps/README.md](deploy/vps/README.md)** and `npm run deploy:vps`.

Live site: https://reportgen.duckdns.org
